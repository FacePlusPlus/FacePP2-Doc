/info/get_session

描述

获取session相关状态和结果

  • 可能的status：INQUEUE(队列中), SUCC(成功) 和FAILED(失败)
  • 当status是SUCC时，返回结果中还包含session对应的结果
所有session都将在计算完成72小时之后过期，并被自动清除。
status返回值为SUCC仅表示成功取得运行结果，实际任务成功与否请根据result内容判断

参数

  参数名 参数说明
必须 api_key App的Face++ API Key
  api_secret APP的Face++ API Secret
  session_id 由/detection或/recognition中的API调用产生的session_id

返回值说明

字段 类型 说明
session_id string 相应请求的session标识符，可用于结果查询
create_time int 任务开始时间，单位：秒
finish_time int 任务结束时间，单位：秒
result object 返回session_id对应的结果内容
status string 可能取值有：INQUEUE(队列中), SUCC(成功) 和FAILED(失败)

调用示例

https://apicn.faceplusplus.com/v2/info/get_session?api_secret=YOUR_API_SECRET&api_key=YOUR_API_KEY&session_id=0a3985c6422aa7044379482da312a3ae
