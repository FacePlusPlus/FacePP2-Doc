/grouping/grouping

描述

给出一个Faceset，尝试将其分类，使得来自同一个人的Face被放在同一类中。

Grouping所花费的时间较长, 因此该调用是异步的，仅返回session_id。

参数

参数名 参数说明
必须 api_key App的Face++ API Key
api_secret APP的Face++ API Secret
faceset_id 或 faceset_name 相应Faceset的id或name。

返回值说明

字段 类型 说明
session_id string 相应请求的session标识符，可用于结果查询
create_time int 任务开始时间，单位：秒
finish_time int 任务结束时间，单位：秒
face_id string 被检测出的每一张人脸都在Face++系统中的标识符
group array 分类结果，来自不同人的face被归到不同的类中
ungrouped array 未归类face对象，无法被归类的face集合
status string 在/info/get_session中，相应session的与状态。
取值为 SUCC(已完成) / FAILED(失败) / INQUEUE(队列中)

调用示例

https://apicn.faceplusplus.com/v2/grouping/grouping?api_secret=YOUR_API_SECRET&faceset_id=c0d0e124a8921f4895658c2419dc8909&api_key=YOUR_API_KEY
