/person/set_info

描述

设置Person的name和tag

参数

参数名 参数说明
必须 api_key App的Face++ API Key
api_secret APP的Face++ API Secret
person_id 或 person_name 相应Person的id或name
可选 name 新的name
tag 新的tag

返回值说明

字段 类型 说明
tag string person相关的tag
person_name string 相应person的name
person_id string 相应person的id

调用示例

https://apicn.faceplusplus.com/v2/person/set_info?api_key=YOUR_API_KEY&tag=new_tag_for_Cage&person_name=NicolasCage&api_secret=YOUR_API_SECRET

JSON 示例

{
    "person_id": "9eb44387923528f97f8545d8bef906db", 
    "tag": "new_tag_for_Cage", 
    "person_name": "NicolasCage"
}
